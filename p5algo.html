1. Function

• Fungsi adalah sebuah blok code yang akan bekerja ketika dirinya dipanggil
• Kita diperbolehkan untuk mengirimkan data ke sebuah fungsi. Data ini dikenal dengan nama parameter
• Fungsi juga mengembalikan (return) data.

4. Arguments

Membuat arguments (atau parameters) pada Python
• Data yang dikirimkan pada sebuah fungsi dapat disebut juga sebagai argument
•Arguments dituliskan setelah nama fungsi dan diletakkan di dalam kurung
• Arguments bisa berjumlah lebih dari satu dan dipisahkan dengan koma

Membuat arguments (atau parameters) pada Python

• Jumlah arguments yang dikirimkan ke fungsi harus berjumlah sesuai dengan jumlah arguments yang didefinisikan ketika membuat sebuah fungsi.

Contoh berikut menunjukkan error ketika jumlah argument yang dikirimkan tidak sesuai dengan jumlah argumen yang didefinisikan.

5. Arbitrary Arguments

Membuat arbitrary arguments pada Python

• Jika Anda tidak mengetahui berapa jumlah arguments yang akan dikirimkan ke dalam sebuah fungsi, tambahkan simbol * di depan argument yang didefinisikan.

6. Keyword Arguments

Membuat keyword arguments pada Python

• Jika Anda ingin mengirimkan data pada sebuah fungsi tanpa ingin memperdulikan urutannya, maka harus mengguna

7. Arbitrary Keyword Arguments

Membuat arbitrary keyword arguments pada Python

• Jika Anda tidak mengetahui jumlah keyword yang akan dikirimkan, maka gunakan ** yang diletakkan di sebelum nama argument ketika didefinisikan.

8. Default Arguments

Membuat default arguments pada Python

• Default arguments digunakan untuk memberikan nilai argument pada sebuah fungsi ketika tidak ada data yang dikirimkan ke fungsi tersebut.
